Proyecto

El TFM busca analizar y predecir el impacto económico de fenómenos naturales extraordinarios en España. La idea central es combinar datos históricos, variables físicas del evento y exposición territorial para construir una base común de análisis.

Pregunta de negocio

Queremos entender qué territorios presentan mayor riesgo económico, qué factores explican mayores pérdidas y cómo construir una base que permita estimar el impacto esperado de futuros eventos.

Arquitectura del proyecto

Datos históricosCNIH, EGIF y CCS aportan eventos, daños e indemnizaciones observadas.

Resumen ejecutivo del análisis
  • Este dataset es el entregable principal del bloque CNIH y sirve como base de trabajo para inundaciones.
  • Cada registro representa una combinación de evento y municipio afectado.
  • El dataset integra información económica, temporal, territorial e hidrológica para facilitar el análisis posterior.
  • El objetivo es que pueda cruzarse con otras fuentes como CCS, GHCN/AEMET, INE, SNCZI/ARPSI y EGIF.
Hallazgos principales
  • El dataset final contiene 7.713 registros y 46 variables.
  • La variable económica principal es indemnizacion_ccs_eur.
  • La clave principal para relacionar tablas CNIH es CE_COD_EVENTO.
  • Se detecta el siguiente gap temporal dentro del rango observado: 2011-2017.
  • Las indemnizaciones presentan una distribución muy concentrada: muchos registros con importes bajos o medios y pocos casos de impacto muy elevado.
  • El ranking territorial permite identificar zonas con mayor daño económico acumulado, aunque todavía no debe interpretarse como ranking definitivo de riesgo.

Limitaciones
  • No todos los eventos históricos tienen información económica municipal.
  • El gap temporal detectado debe tratarse como una limitación antes del modelado.
  • Algunas variables hidrológicas presentan cobertura irregular o valores nulos.
  • CNIH solo cubre inundaciones; el dataset final deberá integrarse con otras fuentes.
