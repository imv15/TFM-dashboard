Resumen ejecutivo

El objetivo del TFM es construir una base analítica para estimar y explicar el impacto económico de fenómenos naturales extraordinarios en España. El proyecto combina fuentes de inundaciones, incendios, indemnizaciones, meteorología y exposición territorial.

Este portal se actualiza automáticamente a partir de los resúmenes generados por los notebooks. Los análisis, rankings y variables clave proceden de archivos JSON internos, no de texto escrito manualmente en el HTML.

Qué hemos hecho para obtener el descargable
  • Partimos de tablas originales del CNIH exportadas desde Access a CSV.
  • Tras un primer análisis, seleccionamoslas tablas útiles para eventos, daños económicos, daños personales y variables hidrológicas.
  • Normalizamos fechas, localización y variables económicas.
  • Unificamos a nivel evento–municipio.
  • El resultado preliminar a la limpieza, está disponible en la pestaña de descargas como el descargable dataset_base_unificado_cnih.csv.Validamos la presencia de variables de calado, caudal y ARPSI mediante indicadores de cobertura.
Resumen ejecutivo del entregable
  • Este dataset es el entregable principal del bloque CNIH y sirve como base de trabajo para inundaciones.
  • Cada registro representa una combinación de evento y municipio afectado.
  • El dataset integra información económica, temporal, territorial e hidrológica para facilitar el análisis posterior.
  • El objetivo es que pueda cruzarse con otras fuentes como CCS, GHCN/AEMET, INE, SNCZI/ARPSI y EGIF.
Hallazgos principales
  • El dataset final contiene 7.713 registros y 49 variables.
  • La variable económica principal es indemnizacion_ccs_eur.
  • La clave principal para relacionar tablas CNIH es CE_COD_EVENTO.
  • Se detecta el siguiente gap temporal dentro del rango observado: 2011-2017.

Limitaciones
  • No todos los eventos históricos tienen información económica municipal.
  • El gap temporal detectado debe tratarse como una limitación antes del modelado.
  • Algunas variables hidrológicas presentan cobertura irregular o valores nulos.
  • CNIH solo cubre inundaciones; el dataset final deberá integrarse con otras fuentes.
