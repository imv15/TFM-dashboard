Resumen ejecutivo

El objetivo del TFM es construir una base analítica para estimar y explicar el impacto económico de fenómenos naturales extraordinarios en España. El proyecto combina fuentes de inundaciones, incendios, indemnizaciones, meteorología y exposición territorial.

Este portal sirve como espacio de seguimiento del trabajo: resume decisiones, muestra el estado de las fuentes y centraliza los datasets preparados para el equipo.

Proyecto

El TFM busca analizar y predecir el impacto económico de fenómenos naturales extraordinarios en España. La idea central es combinar datos históricos, variables físicas del evento y exposición territorial para construir una base común de análisis.

Pregunta de negocio

Queremos entender qué territorios presentan mayor riesgo económico, qué factores explican mayores pérdidas y cómo construir una base que permita estimar el impacto esperado de futuros eventos.

Criterio de incorporación

Una fuente se incorpora al proyecto si aporta al menos una dimensión útil: daño económico, localización, severidad física, meteorología o exposición territorial.
