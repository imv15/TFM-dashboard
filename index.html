CPRO Auxiliar TAux_Provincias Integer Código oficial de provincia (INE). Dos dígitos (01–52). Clave de unión entre todas las fuentes del proyecto: CNIH, AEMET, CCS, INE y SNCZI usan este código como referencia territorial común. Clave de join territorial universal
