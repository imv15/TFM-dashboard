Documento vivo de seguimiento del proyecto: fuentes de datos, decisiones, estado y próximos pasos.

Resumen ejecutivo

El proyecto busca desarrollar una plataforma de análisis y predicción del impacto económico de fenómenos naturales extraordinarios en España. El enfoque actual consiste en integrar varias fuentes públicas: inundaciones, incendios, indemnizaciones históricas, meteorología y variables de exposición territorial.

El análisis no se apoya en una única base de datos. CNIH se trabaja como fuente principal para inundaciones, EGIF será la fuente principal para incendios y CCS funcionará como referencia económica agregada y validación aseguradora.

Detalle CNIH

El CNIH se ha trabajado como fuente principal para inundaciones. La base original estaba en formato Access y se ha exportado a CSV mediante Python.

Ya se ha construido un dataset base a nivel evento–municipio, incorporando información temporal, territorial, económica y de trazabilidad. El siguiente paso es consolidar variables hidrológicas como calado, caudal y ARPSI.

Detalle EGIF

EGIF será la fuente principal para incendios forestales. Aporta registros históricos de incendios, superficie afectada, localización, fechas y variables del siniestro.

Estado actual: pendiente de unificar los ficheros en una única tabla de trabajo.

Detalle CCS

El CCS aporta información económica real vinculada a indemnizaciones por riesgos extraordinarios. Su uso previsto es como validación económica agregada y referencia aseguradora.

Fuentes pendientes

AEMET/NOAA, INE, SNCZI, IGN y Catastro se consideran fuentes de enriquecimiento. No forman parte del núcleo inicial, pero pueden mejorar la calidad del dataset final de modelado.

Principales conclusiones
  • Se ha identificado CE_COD_EVENTO como clave principal para relacionar las tablas del CNIH.
  • La tabla CNIH_DAÑO_ECONOMICO_MUNICIPIO es la fuente económica más útil gracias a la variable CDEM_INDEMNIZACION_CCS.
  • El dataset base integra información temporal, territorial, económica, social y de trazabilidad.
  • La información económica municipal cubre 7.713 registros, correspondientes a 675 eventos y 2.837 municipios.
  • Las indemnizaciones tienen una distribución coherente con fenómenos extremos: muchos importes moderados y pocos eventos de gran impacto.
  • CNIH se considera fuente principal para inundaciones, pero el modelo final deberá integrarse con CCS, EGIF, AEMET/NOAA, INE y otras fuentes territoriales.

Riesgos y limitaciones detectados
  • La información económica municipal solo está disponible para una parte de los eventos totales.
  • Algunas variables hidrológicas de la tabla principal presentan una elevada proporción de nulos.
  • El dataset CNIH solo cubre inundaciones; no debe confundirse con el dataset final del TFM.
  • Será necesario complementar CNIH con EGIF, CCS, meteorología y exposición territorial.
Próximos pasos CNIH
  1. Validar la cobertura real de las variables de calado, caudal y ARPSI tras su incorporación.
  2. Generar una versión estable del dataset base CNIH.
  3. Documentar qué variables pasan al dataset final y cuáles quedan descartadas.
  4. Preparar el análisis de EGIF para construir una base equivalente de incendios.
